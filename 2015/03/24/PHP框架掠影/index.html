PHP框架掠影

Mar 24, 2015

#Yaf

#Laravel
Laravel官网上给自己的定位就是：“The PHP framework for web artisans.”太有情怀了，简直就是PHP框架界的罗玉龙。GitHub上star最多的PHP项目，真的厉害。

#Phalcon
看名字就能猜出来，这个框架是主打速度的。纯C编写，打包成PHP的扩展（咦，好像Yaf也是这样），Phalcon保证了更少的资源消耗和更快的处理速度。

以前看PHP的书时都被教育，对于一个网站应用，性能并不是优先级最高的考虑因素。但是现在Google已经将网站速度作为搜索排名的一个考虑因素了。

#Symfony2
据说学习曲线较陡。不过其实都看了一点它的源代码了……

#Yii
最熟悉的框架了。据说是开发Web2.0的最优框架。也是开发社交网络站点的完美选择。

#CodeIgniter
可能是上手最简单的一个框架了。

#CakePHP

#Aura

#Zend

#Kohana
目标是成为一个很快的PHP框架，但是据说比CodeIgniter还慢。

#Slim
轻量级选手。Benchmark分数据说很高，仅次于Phalcon。
