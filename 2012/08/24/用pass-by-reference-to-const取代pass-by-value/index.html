用pass by reference to const取代pass by value

Aug 24, 2012

缺省情况下，C++以传值方式将对象传入或传出函数。这个特性继承自C。

也就是说，调用函数时，函数的参数以真身的拷贝进行初始化，而函数的调用者收到的也是函数返回值的一个拷贝。拷贝是由对象的拷贝构造函数生成的。所以，pass by value是一项代价很大的操作。
